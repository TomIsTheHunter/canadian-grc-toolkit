# FRFI Compliance Documentation

> This repository contains compliance reference documentation for a federally regulated financial institution (FRFI) in Canada. It covers OSFI mandatory reporting obligations, related regulatory frameworks, and cross-referenced industry standards.

---

## Primary Reference Document

| Document                                                                                     | Description                                                                                                                                                                                                                                                                                                                            |
|----------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [OSFI_Incident_Reporting_Trigger_Criteria.md](./OSFI_Incident_Reporting_Trigger_Criteria.md) | **Core operational reference** — OSFI Technology and Cybersecurity Incident Reporting Advisory (August 2021). Covers trigger criteria (Groups A–D), the 24-hour regulatory deadline, the 12-hour internal escalation threshold, reporting procedure, subsequent reporting, consequences of non-reporting, and cross-framework mapping. |
